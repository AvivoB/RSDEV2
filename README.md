# resdev

### Fonctionnalités :

- Inscription d'un nouvel utilisateur 
- connexion (pas de connexion avec compte google pour l'instant)
- acceder à son profil (incomplet) 
- envoyer des demandes d'amis  
- voir et confirmer les demandes d'amis reçues
- voir la liste des demandes envoyées
- voir la liste des amis
- supprimer des amis


### bugs/problèmes :
- on peut recevoir plusieurs fois la meme demande d'ami 
- certains vues ne sont pas très bien alignés/positionnées (la partie graphique en général n'est pas à point)
- l'inscription d'un utilisateur n'inclut pas encore la date de naissance
